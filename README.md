# Business Performance Dashboard - Power BI

## Project Overview
This project presents a Power BI dashboard designed to monitor commercial performance, monthly reporting and strategic KPIs in a retail/luxury business context.

## Context
The dashboard was created to help business teams track sales performance, analyze market evolution, monitor key indicators and support decision-making through interactive visualizations.

## Tools Used
- Power BI
- DAX
- Excel
- Data cleaning
- Business analysis
- KPI design

## Main Dashboard Pages
- Executive overview
- Sales performance analysis
- Market performance tracking
- Monthly reporting
- Product/category performance

## KPIs Tracked
- Revenue
- Sales evolution
- Average order value
- Market performance
- Product/category performance
- Monthly performance indicators
